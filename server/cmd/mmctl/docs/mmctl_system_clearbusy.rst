mmctl system clearbusy
----------------------

Clears the busy state

Synopsis
~~~~~~~~


Clear the busy state, which re-enables non-critical services.

::

  mmctl system clearbusy [flags]

Examples
~~~~~~~~

::

    system clearbusy

~~~~~~~
