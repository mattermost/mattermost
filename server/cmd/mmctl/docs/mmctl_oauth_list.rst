mmctl oauth list
----------------

List OAuth2 apps

Synopsis
~~~~~~~~


list all OAuth2 apps

::

  mmctl oauth list [flags]

Examples
~~~~~~~~

::

    oauth list

Options
~~~~~~~

::

  -h, --help           help for list
      --page int       Page number to fetch for the list of OAuth2 apps
      --per-page int   Number of OAuth2 apps to be fetched (default 200)
